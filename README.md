# sts3215-node

STS3215 servo controller for Node.js

## Installation

To install dependencies:

```bash
npm install
```

## Development

To run in development mode with hot reload:

```bash
npm run dev
```

To build the project:

```bash
npm run build
```

To run the built project:

```bash
npm start
```

## Testing

To run tests:

```bash
npm test
```

To run tests in watch mode:

```bash
npm run test:watch
```

## Requirements

- Node.js >= 18.0.0
- TypeScript 5.x
